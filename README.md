## Featured Projects

### Delhi_AQI_Project
A comprehensive analysis of Delhi air quality data (PM2.5, PM10, etc.) involving data ingestion, cleaning, exploratory analysis, visualization and presentation generation. Includes scripts for bulk downloading, database creation, plotting and generating polished PowerPoint reports.

### Concrete Strength Prediction using Artificial Neural Networks
A machine learning project using an ANN model to predict the compressive strength of concrete. Contains dataset, training/testing code, model files, and visualizations comparing actual vs. predicted values.

### Stock Market Analytics Project
An analytical study of stock market data implementing various financial indicators and predictive models to assist investment decisions.

### Delhi AQI Research / Thesis Files
Supporting research materials and notebooks related to Delhi air quality studies including empirical equations, model training/testing, and visual presentations.
